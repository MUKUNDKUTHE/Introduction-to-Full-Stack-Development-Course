# Fullstack Auth + Fake News (Static)
Modified version where index.html is directly inside frontend folder.

## Run Instructions
1️⃣ Start MongoDB (localhost:27017)
2️⃣ Backend:
```bash
cd backend
npm install
npm start
```
3️⃣ Frontend:
```bash
cd ../frontend
npm install
npm run dev
```
Then open http://localhost:5173 in browser.
